# miniguia-estudos-notebooklm
Projeto de estudo utilizando o NotebookLM e inteligência artificial.
Esse projeto foi desenvolvido como parte do projeto do desafio DIO utilizando o NotebookLM como ferramenta de apoio aos estudos com Inteligência Artificial.
O objetivo desse repositório é demonstrar como a IA pode auxiliar na organização do conhecimento, criação de resumos, geração de perguntas e apoio ao aprendizado ativo.
Tema escolhido: Phyton para iniciantes.
Objetivos: O objetivo é principal é usar o NotebookLM como ferramnta de estudo e organização, treinando uma IA para que possa fazer pesquisar sobre diversos temas não só phyton, filtrando fontes de pesquisas para evitar alucinações. Usando filtros e links, Pdfs e pesquisas feitas por universidade, podemos usar isso como material de apoio para nossos estudos, criando videos, Pdfs, slides e mapa mental gerado diretamente no NotebookLM.

# miniguia-estudos-notebooklm
Projeto de estudo utilizando o NotebookLM e inteligência artificial.
Esse projeto foi desenvolvido como parte do projeto do desafio DIO utilizando o NotebookLM como ferramenta de apoio aos estudos com Inteligência Artificial.
O objetivo desse repositório é demonstrar como a IA pode auxiliar na organização do conhecimento, criação de resumos, geração de perguntas e apoio ao aprendizado ativo.
Tema escolhido: Phyton para iniciantes.
Objetivos: O objetivo é principal é usar o NotebookLM como ferramnta de estudo e organização, treinando uma IA para que possa fazer pesquisar sobre diversos temas não só phyton, filtrando fontes de pesquisas para evitar alucinações. Usando filtros e links, Pdfs e pesquisas feitas por universidade, podemos usar isso como material de apoio para nossos estudos, criando videos, Pdfs, slides e mapa mental gerado diretamente no NotebookLM.

Exemplo 1 de pergunta ai NotebookLM e sua devida resposta.

Quais bibliotecas de dados e IA são mais recomendadas?

Resposta: NumPy: É considerado o pilar da computação numérica e o motor por trás de muitas outras bibliotecas
Ele é essencial para lidar com arrays e operações matemáticas rápidas, sendo obrigatório para quem foca em dados ou IA

Pandas: Descrita como a "rainha" para manipulação de dados, é ideal para trabalhar com planilhas e tabelas, transformando dados desorganizados em estruturas fáceis de analisar.

SciPy: Utilizada complementarmente para computação científica e técnica.
Visualização de Dados

Matplotlib: Uma das ferramentas mais flexíveis para transformar números em gráficos

Seaborn: Trabalha em conjunto com a Matplotlib, mas foca em criar visualizações mais "estilosas" e de forma mais simples
Inteligência Artificial e Machine Learning

Scikit-learn: Recomendada para quem busca machine learning simplificado, pois oferece algoritmos prontos para tarefas como classificação, regressão e agrupamento (clustering)

TensorFlow, Keras e PyTorch: Formam o "trio poderoso" para Deep Learning (IA avançada).

O Keras e o TensorFlow são frequentemente citados como referências para treinar modelos complexos de inteligência artificial.

As fontes sugerem um caminho evolutivo: comece dominando as bases com NumPy e Pandas para, só depois, avançar para bibliotecas de IA mais robustas como o TensorFlow.
